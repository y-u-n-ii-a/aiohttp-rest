# Demo example: aiohttp, sqlalchemy, marshmallow, s3botocore

## Usage

#### Clone the project
```bash
$ git clone https://gitlab.com/yuniiaolkhova/yalantis-aiohttp.git
$ cd ./aiohttp-rest
```

#### Activate venv and install requirements
```bash
$ make setup
```

#### Run server
```bash
$ make run
```

#### Run linters
```bash
$ make lint
```
